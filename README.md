# BAUST Tea Bar

A comprehensive Tea & Snack Order and Billing Hub designed specifically for BAUST teachers and staff.

## Features
- Complete Order Management & Dispatching
- Real-time Menu Catalogs
- Automated Monthly Billing Ledgers
- Administrative Dashboards & Directory Control
- Firebase Backend Auth & Synchronization

## Launch
1. Ensure dependencies are installed: `npm install`
2. Start development sever: `npm run dev`
